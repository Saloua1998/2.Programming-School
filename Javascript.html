JavaScript

JavaScript is a programming language that adds interactivity to your website. For example the behavior of responses when buttons are pressed or with data entry on forms; with dynamic styling; with animation, etc.

JavaScript is versatile and beginner-friendly. With more experience, you'll be able to create more interactive websites, animated 2D and 3D graphics, comprehensive database-driven websites, and much more!

JavaScript itself is relatively compact, yet very flexible. Developers have written a variety of tools on top of the core JavaScript language, unlocking a vast amount of functionality with minimum effort.
